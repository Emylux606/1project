HTML Foundations Tutorial

Introduction to HTML

HTML stands for HyperText Markup Language. It's used to create the structure and content of web pages.

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>

  <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>


Basic Tags

HTML documents are made up of elements represented by tags. Some common tags include headings (<h1> - <h6>), paragraphs (<p>), links (<a>), images (<img>), and more.

Semantic HTML

Semantic HTML introduces meaning to the web page rather than just presentation. Examples include <header>, <footer>, <nav>, <section>, <article>, and <aside>.

Html foundation compilations

Step-by-step basics of HTML structure. Below are examples of the most common tags you'll use:

Headings

<h1>Main Title</h1>
  <h2>Subtitle</h2>
  <h3>Section Heading</h3>

Paragraphs

<p>This is a paragraph of text explaining something.</p>

Links

<a href="https://example.com">Visit Example</a>

Images

<img src="image.jpg" alt="Description" width="300">

Lists

<ul>
  <li>First Item</li>
  <li>Second Item</li>
</ul>

<ol>
  <li>Step One</li>
  <li>Step Two</li>
</ol>

Tables

<table border="1">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>John</td><td>25</td></tr>
</table>

Forms

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name"><br><br>
  <input type="submit" value="Submit">
</form>




CSS Styling Tutorial

  • CSS Selectors
  • CSS Properties

Introduction to CSS

CSS stands for Cascading Style Sheets. It's used to control the layout and appearance of web pages.

body {
  background-color: #f2f2f2;
  font-family: Arial, sans-serif;
}


CSS Selectors

CSS selectors are used to target specific elements on a web page. Examples include element selectors, class selectors, and ID selectors.

.header {
  background-color: #333;
  color: #fff;
}

CSS Properties

CSS properties are used to define the styles for selected elements. Examples include color, background-color, font-size, and more.

p {
  font-size: 18px;
  line-height: 1.5;
}








Responsive Design Tutorial

  • Introduction to Responsive Design
  • Media Queries
  • Flexible Grids


Introduction to Responsive Design

Responsive design is an approach to web design that makes web pages render well on a variety of devices and screen sizes.

Media Queries

Media queries are used to apply different styles based on different screen sizes or devices.

@media (max-width: 768px) {
  .container {
    padding: 20px;
  }
}

Flexible Grids

Flexible grids are used to create layouts that adapt to different screen sizes.

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

Introduction to JavaScript

JavaScript is a programming language used to add interactivity to web pages.

console.log('Hello, World!');

Variables and Data Types

Variables are used to store values, and data types determine the type of value a variable can hold.

let name = 'John';
const age = 30;

Functions

Functions are reusable blocks of code that perform a specific task.

function greet(name) {
  console.log(`Hello, ${name}!`);
}
